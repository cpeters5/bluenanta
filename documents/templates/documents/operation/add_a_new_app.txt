
To add a new application:
-  python manage.py startapp <newapp name>
-  add <newapp> to INSTALLED_APPS
-  add newapp.urls.py path to the main urls.py
-  Copy medels.py, url.py, views.py, form.py from other app
-  edit newapp.urls.py to change app name to newapp.
-  edit newapp.forms.py to change <oldapp>.models to <newapp>.models

DATA
-  edit models.py and change all related_name to a unique name
-  python manage.py makemigrations <newapp>
-  python manage.py migrates
-  redesign each <newapp>_* tables.  Check all constraits, triggers, datetime.
-  If some <newapp> data already exists in other app (e.g. other app), move them to <new app> tables

FUNCTIONS
-  NAVBAR: Add a button to second navbar for the newapp --> browsegen <newapp>
-  HOME PAGE: Add sample images of the <newapp> to home page
-  SEARCH:
    - Add search_<newapp> method to search.views (copy from a <oldapp>_search method and change app name.)
    - Add a url path to search.urls.py
