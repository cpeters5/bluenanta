Create a new application:
Process
  • python manage.py startapp newapp name
  • add newapp to INSTALLED_APPS
  • add newapp.urls.py path to the main urls.py
  • Copy medels.py, url.py, views.py, form.py from other app
  • edit newapp.urls.py to change app name to newapp.
  • edit models.py and change all related_name to a unique name
  • python manage.py makemigrations newapp
  • python manage.py migrates
  • redesign each newapp_* tables. Check all constraits, triggers, datetime.
  • If some newapp data already exists in other app (e.g. other app), move them to newapp tables
  • edit newapp.forms.py to change oldapp.models to newapp.models
